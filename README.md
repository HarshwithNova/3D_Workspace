🎮 How It Works

The webcam captures hand movements.

MediaPipe detects hand landmarks in real-time.

Gesture data is sent to the particle system.

The Three.js engine renders 8000 particles responding to gestures.

Gesture examples:

Gesture	Result
Open hand	Flower particles bloom
Circular motion	Buddha pattern forms
Fast swipe	Fireworks explosion
Continuous motion	Color shifting particles
🧠 Performance Optimization

To handle 8000 particles smoothly, the system uses:

GPU based particle rendering

Instanced geometry

Optimized shaders

Efficient memory buffers

This ensures high FPS even on laptops.
